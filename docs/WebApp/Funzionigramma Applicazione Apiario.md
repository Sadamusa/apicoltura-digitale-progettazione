Funzionigramma Applicazione Apiario

-Funzioni dell’interfaccia Web App
 
1. Interfaccia di accesso:
-   F1.0- Login: Permette l’accesso alla Web App tramite l’inserimento di una API Key personale associata al database dell’utente. se l’utente è già registrato vai alla schermata apiario.
-   F1.1- Logout: Permette la disconnessione dell'account.
2. Interfaccia Home:

-   Schermata Apiario:

-   F2.0-Selettore Apiario: Visualizzazione a schede degli apiari disponibili.Selezionando una scheda è possibile visualizzare nella dashboard i dati relativi all’arnia scelta. caricando i dati di quella arnia nella dashboard principale.
-   F2.1-Bottone Impostazione: Apertura sezione taratura sensori con i vari sensori e combobox per modificare i valori da tarare e bottone salva.
-   F2.2-Aggiungi Arnia:  Permette di aggiungere una nuova arnia all’apiario selezionato. e cliccandoci sopra alla arnia aggiunta è possibile vedere nella dashboard i dati relativi a quella arnia.

-   Dashboard principale
-   F3.0-Sezione Grafici: Visualizzazione dinamica dei grafici relativi a:peso,umidità,temperatura
-   F3.1-Sezione Sensori: Mostra i dati dei sensori in tempo reale, aggiornati automaticamente dal sistema, il database carica le informazioni nelle label di questa sezione.
-   F3.2-Sezione Notifiche: Area dedicata alle notifiche e agli avvisi, con popup di allerta relativi allo stato dei sensori e agli eventi critici dell’arnia.
-   F3.3- Grafico filtrato: Possibilità di filtrare i dati nel grafico, peso, temperaura ,umidità, errori.

-   Dashboard sezione taratura:
-   F4.0- Bottone Salva sensore Umidità.
-   F4.1- Bottone Salva sensore Tempertura.
-   F4.2 - Bottone Salva sensore Peso.
